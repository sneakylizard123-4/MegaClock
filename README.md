## Parts
An alarm clock made with the Arduino Mega 2560 using the DS1307 RTC module
Plastic standoffs until case is made
Buzzer plugged directly into headers

### What Does it Do?
Wake up at set time with neopixels and a buzzer
blinking and beeping for maximum awakeness
